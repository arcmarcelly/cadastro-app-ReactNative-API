# Cadastro App React Native consumindo API

Aplicativo mobile desenvolvido em React Native com Expo para cadastro de usuários integrando com API ViaCEP.

---

## 📱 Funcionalidades

- Cadastro de usuário
- Validação de formulário
- Validação de email
- Validação de senha
- Confirmação de senha
- Busca automática de endereço pelo CEP
- Integração com API ViaCEP
- Interface moderna e responsiva
- Loading durante busca do CEP
- Tratamento de erros

---

## 🛠 Tecnologias Utilizadas

- React Native
- Expo
- JavaScript
- ViaCEP API
